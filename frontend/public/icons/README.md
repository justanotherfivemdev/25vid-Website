# PWA Icons

Place the following icon files in this directory:

- `icon-192.png` — 192×192 PNG (required for Android install)
- `icon-512.png` — 512×512 PNG (required for Android splash screen)
- `icon-maskable-512.png` — 512×512 PNG with safe zone padding (40% center)

Generate from the 25th ID patch image. Use a tool like:
- https://maskable.app/editor to create the maskable variant
- https://realfavicongenerator.net for full icon set

The background should be `#050a0e` (mil-bg-deep) with the 25th ID patch centered.
